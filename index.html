Gravitational wave analysis with LISA data for classification of sources from the signal. By using Bayesian statistic and machine learning models I aim to classify sources between double white dwarf or neutron star white dwarf binaries. Employed different supervised learning models such as k-NN, SVM, Random Forest , XGBoost, NGBoost and neural network to compare the classification performance. By using bayesian classification, we put the probabilities produced by the machine to be a better representation of the real events. By employing Bayesian Optimisation, we aim to tune the hyperparameters of the classification model to be optimised for our dataset.
